# 儿童识字小报生成器

一款帮助家长和老师生成儿童识字小报的 AI 提示词工具。

## 功能特点

- **智能提示词生成** - 输入主题和标题，自动生成符合儿童认知的 AI 绘图提示词
- **14+ 预设场景** - 超市、医院、公园、学校、餐厅、家庭、图书馆、游乐场、动物园、海滩、火车站、消防站、银行、花园
- **一键复制** - 生成的提示词支持一键复制
- **图像生成**（可选）- 支持直接调用 Nano Banana Pro API 生成图像

## 使用方法

### 1. 打开应用

直接在浏览器中打开 `index.html` 文件。

### 2. 选择或输入主题

- 点击预设主题按钮快速选择
- 或手动输入自定义主题

### 3. 生成提示词

点击「生成提示词」按钮，获取完整的 AI 绘图提示词。

### 4. 复制或生成图像

- 点击「复制提示词」将提示词复制到剪贴板
- 配置 API Key 后，可直接生成图像

## API 配置（如需生成图像）

1. 访问 [Nano Banana Pro API](https://kie.ai/api-key) 获取 API Key
2. 在页面中展开「API 配置」，输入 API Key 并保存
3. 设置图像参数（比例、分辨率、格式）
4. 点击「生成图像」
